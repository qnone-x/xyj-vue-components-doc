# 获取当前时间
```js
  import {currentTime} from 'xyj-vue-components/utils';

<template>  
       <div>
              {{currentTime('yyyy-MM-dd hh:mm:ss')}}
       </div>
       <div>
              {{currentTime('MM-dd hh:mm:ss')}}
       </div>
       <div>
              {{currentTime('yyyy-MM-dd')}}
       </div>
</template>
```
